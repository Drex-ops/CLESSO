This folder contains large data files required for the TERN biodiversity index analysis. These files are excluded from version control (see `.gitignore`).

The following data files should be placed in this directory to run the analysis:

- `transfer_3496189_files_dc31bd16.tar` - Compressed data archive

1. Download or prepare the required data files
2. Place them in this folder
3. Extract archives as needed
4. Update this index as you add new datasets
